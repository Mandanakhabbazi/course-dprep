## Overview 
Distribute your data across multiple tabs (alternative to a sidebar layout).

## Code 

```R
tabsetPanel(
	tabPanel(title = "tab 1", 
    h1("Overview"),
    "Content goes here"),
	tabPanel(title = "tab 2", "The content of the second tab"),
	tabPanel(title = "tab 3", "The content of the third tab")
)
```
